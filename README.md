# Interactive To-Do (React + TypeScript)

A focused to-do app with optional due date/time, live clock, filters, and `localStorage` persistence with runtime validation.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+ (includes npm)

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Push to GitHub

From this folder (`todo-web`), with a new empty repo created on GitHub:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

If you use SSH:

```bash
git remote add origin git@github.com:YOUR_USERNAME/YOUR_REPO.git
```
